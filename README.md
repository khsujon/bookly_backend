# Bookly Backend

A RESTful API backend for a book review application built with FastAPI.

## Tech Stack

- **Python 3.x**
- **FastAPI** - Modern web framework
- **Pydantic** - Data validation
- **Uvicorn** - ASGI server

## Setup

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd Bookly_backend
   ```
